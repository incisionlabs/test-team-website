# Demo team website

Built with Jekyll, managed with DecapCMS, hosted on Netlify.

## Edit Content

**Via CMS (Preferred):**
- Go to `/admin/` on your live site
- Click "Blog" to add/edit blog posts
- Upload images if required
- Changes auto-save to GitHub

**Via GitHub:**
- Edit `.md` files directly
- Blog posts in `_posts/` folder
- Format: `YYYY-MM-DD-title.md`

## Replace Placeholder Images

- **Current:** Using Unsplash URLs (free stock photos)
- **To replace:** Upload your images via CMS or add to `/assets/images/`

## Customization

- **Colors/fonts:** Edit `assets/css/style.css`
- **Site title:** Edit `_config.yml`
- **Navigation:** Edit `_layouts/default.html`
